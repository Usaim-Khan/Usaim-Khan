## Featured Project

### [PracticeLog API](https://github.com/Usaim-Khan/Practice-Log)

A RESTful backend for tracking coding practice — Topics, Problems, and Attempts — with 15 CRUD endpoints, enforced referential integrity at the application layer, and a bundled vanilla-JS dashboard served from the same FastAPI instance.

**Stack:** FastAPI · PostgreSQL · SQLAlchemy 2.0 · Pydantic v2

🔗 [Live demo](https://practice-log.up.railway.app/) · [API docs](https://practice-log.up.railway.app/docs)
